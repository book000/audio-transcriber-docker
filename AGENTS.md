# AGENTS.md

## 目的
このドキュメントは、エージェント共通の作業方針を定義する。

## 基本方針
- 会話言語: 日本語
- コメント言語: 日本語
- エラーメッセージ言語: 英語
- コミット規約: Conventional Commits
- 日本語と英数字の間には半角スペースを挿入する

## 判断記録のルール
- 判断内容、代替案、採用理由、前提条件、不確実性を明示する。

## 開発手順（概要）
1. プロジェクト理解: 既存のコード、Dockerfile、README を確認する。
2. 環境準備: Docker を使用したビルドと実行。
3. 変更実装: 既存のスタイルを尊重したコード修正。
4. 検証: 動作確認。

## セキュリティ / 機密情報
- 認証情報や機密情報をコードに含めない。
- ログに個人情報や機密情報を出力しない。

## リポジトリ固有
- Puppeteer による Chrome 自動操作と Web Speech API の利用が核となるプロジェクトである。
- Docker コンテナ内での Pulseaudio 設定が動作に直結する。
